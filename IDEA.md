# 0. 数据来源层

0.1 核心数据源
- ERP系统：采购订单、入库单、发票、供应商主数据
- SRM系统：供应商绩效数据、合同信息
- WMS系统：库存水位、出入库记录
- MES系统：生产消耗、物料需求
- 财务系统：预算、成本、付款

0.2 外部数据源
- 市场价格：我的钢铁网、卓创资讯、中数在线
- 期货价格：大宗商品交易所
- 宏观指标：汇率、关税、PPI/PMI/CPI

0.3 数据更新频率
- 实时数据：库存水位、市场价格
- 日更数据：采购订单、入库记录
- 周更数据：供应商绩效
- 月更数据：预算执行、成本分析

# 核心痛点：

核心痛点
1. 手工报表效率低下
Excel为主，数据收集耗时，分析滞后
2. 数据孤岛严重
多套ERP数据不互通，难以全局视角;
采购系统、预算系统，多系统口径不一
3. 决策缺乏数据支撑
凭经验决策，难以量化降本效果
集团采购总监需要数据的时候，一般是请集团采购部门的同事，从各个系统拉取数据，再进行匹配、汇总
生产材料采购价格与大宗原材料的市场行情不能自动关联分析
4. 风险预警滞后
供应风险发现晚，被动应对

## 我的想法
1. 支出分析（找出降本机会）
1.1 支出的核心品类（化工、钢材、塑料粒子）
1.1.1 支出金额的走势与市场价格的走势对比分析 - 市场价格偏差度分析
1.1.2 支出金额的走势与库存水位的走势对比分析 - 降低库占资金的机会
1.1.3 VMI的品类占比 - 降低库占资金的机会
1.1.4 支出金额在各个供应商的份额分布 - 供应商集中度分析
1.1.5 品类TCO成本的构成（物流成本、仓储成本、税费）分析
1.1.6 长约占比（战略型寻源） - 锁定价格波动
1.1.7 各事业部、各子公司的库存水位与下单走势对比（分品类分析）
1.1.8 结合价格走势和库存水位，基于假设场景，建议采购下单数量

2. 价格预测
2.1 预判价格走势：提前1-3个月预测关键物料价格
2.2 提前布局锁定低价：在价格低点签订长期合同
2.3 优化采购节奏：动态调整采购批量与时机
（历史采购价格、大宗商品市场价格及期货、汇率波动、关税政策、PPI/PMI/CPI指数、季节性因素）

3. 需求预测
3.1 通过需求预测，优化库存水平，降低库存资金占用
3.2 需求聚合与采购杠杆分析



4. 其他降本机会
4.1 小额分散
4.1.1 小额分散订单的供应商整合和集采的机会

4.2 供应源分析（找出优质供应源）
4.2.1 评估供应商绩效与支出匹配度
4.2.2 供应商过多的问题：识别大于5家供应商的物料与供应商清单

- 合规风险：资质过期、审计问题

- 退出供应商建议

# 8. 异常检测与预警

8.1 价格异常预警
- 采购价格偏离市场均价 > X%
- 同一物料不同供应商价格差异 > X%
- 历史价格突增/突降

8.2 支出异常预警
- 预算执行率偏离目标 > X%
- 某品类/事业部支出突增
- 集中度异常（供应商过于集中/分散）

8.3 供应商异常预警
- 绩效评分下降 > X分
- 交付及时率跌破阈值
- 独家供应物料清单

8.4 库存异常预警
- 库存周转天数异常
- 呆滞库存预警
- 缺货风险预警
