# AI 助手生产放行检查清单

版本: v0.1
更新时间: 2026-04-11
适用范围: AI 助手从非生产验证进入真实数据试运行或生产发布前

## 放行门槛

- [ ] 已获取并归档 Coze 平台持久化存储边界结论。
- [ ] 已确认如存在持久化存储，其存储周期与删除机制。
- [ ] 仅在安全专家复核通过后接入真实生产数据。
- [ ] 非生产阶段仅允许测试数据或脱敏数据。

## 环境与配置检查

- [ ] `JWT_SECRET` 使用仓库外安全配置，不使用示例占位值。
- [ ] `CORS_ORIGIN` 已收紧为目标部署域名。
- [ ] `NON_PRODUCTION_ONLY` 在生产放行前已按审批结果调整。
- [ ] `AI_PRIVACY_GATE_REQUIRED=true`，前端首次使用必须点击“同意并使用”。

## 数据与日志检查

- [ ] 日志不记录原始问题全文。
- [ ] 日志不记录完整 KPI 或完整预警快照。
- [ ] 非生产日志仅包含测试数据或脱敏数据。
- [ ] 关键链路保留 `trace_id`、模块、耗时、降级状态。

## 功能与回退检查

- [ ] `/api/ai-proxy/health` 健康检查通过。
- [ ] 降级路径已验证，前端可展示简版结果。
- [ ] 隐私门槛在首开、拒绝、同意三种路径下行为正确。
- [ ] 发布前再次执行门禁文档与本清单的逐项复核。

## 说明

- 当前项目仍处于“带条件放行”阶段。
- 在 Coze 平台持久化存储边界未确认前，不允许真实生产数据放量。
