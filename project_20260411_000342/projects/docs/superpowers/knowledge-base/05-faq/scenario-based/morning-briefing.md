# 晨会追问场景

版本: v0.1
更新时间: 2026-04-11

## 场景 1：总监追问降本口径

问题: 你刚才说本月降本达成率 81.1%，这个降本额包含物流费用吗？

建议回答:

- 先给结论: 不包含物流费用。
- 再给解释: 当前降本额只按采购单价差计算。
- 再给来源: cost-calculation.md。

## 场景 2：总监追问全成本口径

问题: 如果我要看全成本影响，是不是应该看 TCO？

建议回答:

- 先给结论: 是，TCO 更适合看采购到交付的综合成本。
- 再给解释: TCO 包含物流、仓储、关税，不包含返利。
- 再给来源: tco-boundary.md。

## 场景 3：总监追问市场价可信度

问题: 钢材偏离的市场价基准靠谱吗？多久更新一次？

建议回答:

- 先给结论: 当前基准来自 Mysteel、百川与内部台账交叉校验。
- 再给解释: 默认周度更新，高波动品类周中补充复核。
- 再给来源: price-benchmark-rule.md。

## 场景 4：总监追问预警闭环

问题: 这条预警谁负责？什么时候恢复？

建议回答:

- 先给结论: 每条 P0 预警必须包含 Owner、Status 与 ETA。
- 再给解释: 没有责任人或 ETA 不能视为已进入闭环追踪。
- 再给来源: alert-disposition-rule.md。

## 场景 5：总监追问上期单价定义

问题: 你这里的上期单价到底是上个月还是去年同期？

建议回答:

- 先给结论: 要看当前分析口径，月度环比用上月均价，同比用去年同期均价。
- 再给解释: 只有同物料、同规格、同币种并完成换算的数据才可比。
- 再给来源: baseline-price-definition.md。

## 场景 6：总监追问 ETA 来源

问题: 这条供应商 ETA 是从哪来的？

建议回答:

- 先给结论: 当前以 ERP 或 SRM 中采购订单行的确认到货字段为准。
- 再给解释: 口头承诺或群消息如果未回写系统，不纳入口径。
- 再给来源: delivery-commitment-source.md。
