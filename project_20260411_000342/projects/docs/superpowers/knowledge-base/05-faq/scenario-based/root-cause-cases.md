# 根因分析场景案例

版本: v1.0
更新时间: 2026-04-11
适用场景: 晨会追问、异常归因、AI 助手二次解释

## 场景 1：支出异常上升

问题: 为什么本周新能源事业部支出同比突然增加？

建议回答:

- 先判断是价格、采购量还是结构变化驱动。
- 再核对预算执行率与大额订单释放情况。
- 如需解释预算进度，引用 budget-execution.md。

## 场景 2：价格偏离扩大

问题: 为什么钢材采购价偏离市场价 5.8%？

建议回答:

- 先确认市场价来源和更新时间是否仍有效。
- 再看是否存在锁价失效、临采补货或规格不一致。
- 如需解释映射关系，引用 category-commodity-mapping.md。

## 场景 3：库存风险升高

问题: 为什么塑料粒子缺货风险升到 12.8 天？

建议回答:

- 先判断是需求爬升、到货延迟还是安全库存参数偏低。
- 再核对 ETA 是否已回写系统并检查替代料方案。
- 如果需要说明 ETA 来源，可继续引用 delivery-commitment-source.md。

## 场景 4：供应商表现下滑

问题: 为什么某供应商交付及时率跌到 87.2%？

建议回答:

- 先区分是单次事件还是连续趋势下滑。
- 再检查产能、排产、运输异常与异常关闭效率。
- 如果需要明确责任人、状态和 ETA，可继续引用 alert-disposition-rule.md。

## 场景 5：材料成本率恶化

问题: 为什么材料成本率上升，但采购降本额也在增长？

建议回答:

- 先说明两者衡量对象不同，一个看采购价差，一个看材料成本占收入比例。
- 再分析是否因为收入下降、产品结构变化或高成本材料占比提高。
- 如需解释指标定义，引用 savings-amount.md 和 material-cost-rate.md。
